# Images Directory

Ce dossier contient les images utilisées dans l'application.

## Instructions

1. Placez votre image d'arrière-plan pour la page de connexion ici
2. Nommez l'image `background-login.jpg` ou `background-login.png`
3. Le composant Login utilisera automatiquement cette image

## Formats supportés

- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## Taille recommandée

- Dimensions: 1920x1080px ou supérieur
- Poids: < 2MB pour un chargement optimal
